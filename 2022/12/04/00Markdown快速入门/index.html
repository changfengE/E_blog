ctrl + D 选中当前词 ctrl + L 选中当前句/行

ctrl + O 打开文件 ctrl + P 搜索文件并打开
ctrl + 回车 表格下方插入行 ctrl + , 打开偏好设置
ctrl + . 切换全角/半角标点 ctrl + / 切换正常/源代码视图
ctrl + shift + - 缩小视图缩放 ctrl + shift + + 增大视图缩放
